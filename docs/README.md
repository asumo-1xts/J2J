## Concept / Spec

テンキーレスをそのまま2つに割ったような、シンプルな分割キーボード

|                 |                            |
| --------------- | :------------------------- |
| 配列            | US ＋ 変換キー，無変換キー |
| Enterキー       | ISO                        |
| 接続方法        | 有線のみ（USB Type-C）     |
| ホットスワップ  | 対応                       |
| リマッピング    | 対応（Vial）               |
| LEDライティング | 非対応                     |
